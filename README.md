# OrigaMe (Origami Study Hub)

OrigaMe là một Website về origami.

Yêu Cầu Hệ Thống
IDE: PhpStorm
CSDL: MySQL


Cấu hình CSDL
  - Tạo một cơ sở dữ liệu mới trong MySQL.
  - Điều chỉnh các thông tin kết nối CSDL trong file phpOrigamiProject/models/connectDb.php
